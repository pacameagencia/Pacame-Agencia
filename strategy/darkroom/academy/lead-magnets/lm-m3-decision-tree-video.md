## 5 preguntas que cierran la decisión

### Pregunta 1 · ¿Tienes archivo real (grabación) o no?

```
SÍ (cliente envió clips, grabaste algo) → EDIT CLÁSICO base
NO (solo brief, idea, concept)         → IA GENERATIVA base
```

### Pregunta 2 · ¿Cuál es el formato final?

```
Reel vertical 9:16 <60s          → CapCut (edit clásico) o Higgsfield Soul Video (IA)
Pieza horizontal 8-30s hero      → Premiere + DaVinci (clásico) o Cinematic Studio (IA)
Vídeo largo >2 min               → Premiere obligatorio
Story 5s daily                   → CapCut o Higgsfield Soul Video
```

### Pregunta 3 · ¿Cuál es el tier?

```
AUTHENTIC (iPhone-look, daily, batch unlimited)
  → Higgsfield Soul Video + CapCut

HYBRID (algo cinemático pero no hero)
  → Kling 3.0 o base AUTHENTIC + grading DaVinci

CINEMATIC (hero, ad, portfolio)
  → Seedance 2.0 / Cinematic Studio V3 / Veo 3.1 + Premiere + DaVinci
```

### Pregunta 4 · ¿Cuántos shots / escenas distintas?

```
1 shot continuo            → 1-act 8s
2 shots chained            → 2-act 14s con last frame chaining
3-4 shots conectados       → 3-act 20-30s trailer (Hook + Content + CTA)
4+ shots desconectados     → 4 piezas SEPARADAS (NUNCA 4 escenas en 14s · anti-patrón concept 007)
```

### Pregunta 5 · ¿Cuánto cobra el cliente?

```
<100€        → AUTHENTIC only · Higgsfield Soul Video + CapCut
100-500€     → HYBRID · Kling o AUTHENTIC + edit
500-2.000€   → CINEMATIC · Seedance o Cinematic Studio + Premiere + DaVinci
2.000€+      → CINEMATIC top · Veo 3.1 o cam real + post profesional
```

## Las 5 reglas duras de M3

1. **NO mezcles tiers en una sola pieza**. Una pieza = un tier. Si tu prompt mezcla "shot on iPhone" + "shot on 70mm anamorphic", la pieza sale confusa.

2. **NUNCA generes cinematic sin research-first**. 5 datos verificables antes de tocar Higgsfield/Seedance: peli, DP, lentes, LUT, ritmo. Sin esto = créditos gastados.

3. **NO 4 escenas en 14s**. Si tu idea requiere 4 momentos distintos, son 4 piezas 1-act separadas, no 1 trailer condensado (anti-patrón concept 007).

4. **Last frame chaining para 2-act y 3-act**. El último frame de shot 1 es el start frame de shot 2. Sin esto = jump cut feo.

5. **3 variations mínimo siempre**. Nunca generes 1 sola y te conformes. El 80% de la calidad viene de elegir entre 3.
