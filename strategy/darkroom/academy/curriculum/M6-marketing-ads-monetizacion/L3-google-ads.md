# M6.L3 — Google Ads · search + Performance Max

> **Dura**: 12 min
> **Nivel progreso**: 89% → 91%
> **Requisito previo**: M6.L2

## Qué vas a sacar de aquí

Decides cuándo Google Ads supera a Meta/TikTok. Configuras tu primera campaña Search por keyword intent. Conoces Performance Max para captura automatizada multi-canal.

## El concepto (1 idea, no 5)

Google Ads sirve para 2 tipos de captura:

1. **Search** · cliente busca activamente ("comprar mate térmico para correr"). Alta intención. Conversion alta. CPC más alto.
2. **Performance Max (P-Max)** · campaña automática IA Google que muestra ads en Search + YouTube + Display + Discover + Maps + Gmail. Cobertura amplia. Menos control.

Cuándo SÍ Google Ads:

- Producto / servicio con búsqueda activa (cliente sabe lo que quiere).
- Mercado local (restaurante, peluquería, abogado en X ciudad).
- Reservas / citas (clínica dental, consultora).
- Ecommerce con keywords longtail rentables ("paraguas anti-viento profesional").

Cuándo NO Google Ads (mejor Meta/TikTok):

- Producto que el cliente no busca todavía (descubrimiento social).
- Servicios B2C lifestyle (impulso emocional, no búsqueda racional).
- Budget bajo (<$500/mes · Google requiere más volumen para optimizar).

## El ejemplo real

**Caso A · Cliente consultor financiero busca leads**

Estrategia: Google Search Ads.

- Keywords: "consultor financiero para autónomos", "asesor fiscal Madrid", "ayuda renta autónomos".
- Bid medio: €2-5 por clic.
- Conversion (lead form): 8-15%.
- Coste por lead: €15-35.

Funciona porque cliente busca activamente. Meta/TikTok no llegan a esta intención.

**Caso B · Tienda dropshipping mate térmico**

Estrategia: Performance Max + retargeting display.

- Sube catalogo Shopify a Google Merchant Center.
- Configura P-Max con creative variants (foto + video + texto).
- Google IA optimiza distribución automática.
- ROAS objetivo: 2x.

Funciona porque hay búsqueda activa "mate térmico runner" + display visual + YouTube preroll.

**Caso C · Servicio agencia freelance**

Estrategia: Search + landing page.

- Keywords: "agencia branding Madrid", "diseñador freelance Barcelona", "web Shopify a medida".
- Bid: €3-8/clic.
- Lead form en landing · CPC €5, conversion 5%, CPL €100.
- Si ticket medio cliente es €1.500, CPL €100 = excelente.

## El prompt copiable

Setup checklist primera campaña Google Ads:

```
PRODUCTO/SERVICIO: __________
TIPO CAMPAÑA: ☐ Search · ☐ Performance Max · ☐ Display retargeting

KEYWORDS RESEARCH (Google Keyword Planner gratis):
  ☐ 10 keywords longtail con volumen 100-1.000/mes
  ☐ Match types: Phrase match (mejor para empezar)
  ☐ Negative keywords (excluye búsquedas no relevantes)

AD COPY (Search):
  ☐ Headline 1 (≤30 chars): keyword + beneficio
  ☐ Headline 2 (≤30 chars): diferencial
  ☐ Headline 3 (≤30 chars): CTA acción
  ☐ Description 1 + 2 (≤90 chars cada uno)

LANDING PAGE:
  ☐ Página específica (no homepage)
  ☐ Heading coincide con keyword
  ☐ CTA claro arriba del fold
  ☐ Trust signals + reviews

BUDGET inicial:
  ☐ $20-50/día durante 14 días
  ☐ Conversion tracking activo (Google Tag Manager)
```

## Tu ejercicio (5 min)

Si tienes producto / servicio que la gente busca activamente:

- [ ] Google Keyword Planner: encuentra 10 keywords longtail.
- [ ] Anota volumen búsqueda + competencia + CPC sugerido.
- [ ] Calcula: si CPC es €2 y conversion 5%, CPL = €40. ¿Rentable según tu ticket medio?

Si tu producto NO se busca todavía (descubrimiento): Google Ads no es prioridad. Salta a M6.L4 copy o M6.L5 landing.

## Quick-win

**Regla "negative keywords desde día 1"**: lista las búsquedas que NO quieres pagar. Ej. si vendes mate térmico, exclúyelas: "gratis", "barato", "DIY", "fabricar". Sin negative keywords pagas clics que nunca compran. Acumulan 30-50% del gasto sin retorno.
