# M2.L4 — Canva Pro · cuándo sí y cuándo Figma

> **Dura**: 10 min
> **Nivel progreso**: 21% → 23%
> **Requisito previo**: M2.L3

## Qué vas a sacar de aquí

Decides Canva vs Figma en 10 segundos según el encargo. Montas un carrusel IG en Canva en 15 minutos. No vuelves a usar Canva para diseño de sistema (porque no es para eso).

## El concepto (1 idea, no 5)

Canva y Figma resuelven problemas distintos. No es "uno es mejor". Es "cada uno para lo suyo":

**Canva gana cuando**:
- Plantilla rápida (carrusel, post IG, story, infografía suelta).
- No esperas que el diseño se mantenga en el tiempo.
- Cliente no técnico que va a editar después.
- Volumen alto de piezas similares con plantilla repetible.

**Figma gana cuando**:
- Diseño de sistema (landing web, app, brand book).
- Cliente vuelve en 3-6 meses pidiendo cambios.
- Equipo colabora simultáneo.
- Componentes reutilizables son crítico.

Regla práctica: si te van a pedir el archivo editable más adelante para mantener, Figma. Si es entrega "estática" único uso, Canva.

## El ejemplo real

**Cuándo Canva** (3 casos reales):

1. Carrusel 7 slides IG para post semanal del cliente · tema "5 tips marketing 2026" → Canva (template + edit + export 7 PNG).
2. Story para promo flash (descuento 24h) → Canva.
3. Infografía para newsletter quincenal · siempre cambia el dato pero el formato es el mismo → Canva.

**Cuándo Figma** (3 casos reales):

1. Landing web cliente · responsive · cliente revisará en 2 meses para ajustes → Figma.
2. Brand book con paleta + tipos + componentes reutilizables → Figma.
3. App móvil mockup con 12 pantallas relacionadas → Figma.

**Cuándo NINGUNO de los dos**:

1. Logo vector limpio para imprenta → **Illustrator** (vector real, no estilo embebido).
2. Editar foto producto → **Photoshop**.
3. Generar imagen desde cero → **Higgsfield Soul / Nano Banana**.

## El prompt copiable

Workflow Canva para "carrusel 7 slides en 15 min":

```
1. Canva → Template → "Instagram Post" 1080x1080 → 7 frames.
2. Aplica template existente (filtro por nicho/estilo). Edita texto.
3. Pega tu paleta de cliente: Brand Kit → Colors → añade 4 colores.
4. Reemplaza tipografía: Text → Font → busca tipo aprobado por cliente.
5. Aplica color/font a todos los frames (Apply to all pages).
6. Reemplaza imágenes: tus assets ya en Brand Kit · drag&drop sobre frame.
7. Revisa orden: panel pages · arrastra para reordenar.
8. Download → PNG → 7 archivos.
9. Sube a IG carrusel.
```

Tiempo: 15 min para 7 slides decentes. Sin Canva (en Figma o Photoshop): 60-90 min mínimo.

## Tu ejercicio (5 min)

Abre Canva Pro:

- [ ] Crea nuevo proyecto "Instagram Post 1080x1080".
- [ ] Aplica un template del nicho que te interese (busca "minimalist quote post").
- [ ] Cambia el texto a algo tuyo (frase tipo "Cero promesas imposibles").
- [ ] Cambia colores a una paleta (dark + dorado, por ejemplo).
- [ ] Descarga PNG.

Si lo haces en 5 min, ya manejas Canva para 80% de uso.

## Quick-win

**Regla "Canva con Brand Kit antes de empezar"**: si vas a usar Canva con varios clientes, crea Brand Kit por cliente (colores + fuentes + logos). Cuando entras a un proyecto cliente, aplicas Brand Kit con 1 clic. Ahorras 5 min por pieza. En 1 año son 50+ horas.
