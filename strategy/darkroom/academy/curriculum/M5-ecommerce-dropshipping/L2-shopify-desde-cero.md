# M5.L2 — Shopify desde cero · tu primera tienda en 60 min

> **Dura**: 15 min
> **Nivel progreso**: 72% → 74%
> **Requisito previo**: M5.L1

## Qué vas a sacar de aquí

Montas tienda Shopify funcional en 60 minutos: dominio, theme, productos, pago, envío. Tienes el setup para empezar a probar producto real con ads.

## El concepto (1 idea, no 5)

Shopify Basic ($29/mes) cubre 95% de tiendas dropshipping. Setup mínimo viable:

1. **Cuenta + dominio** · crea cuenta Shopify · conecta dominio propio (no usar .myshopify.com).
2. **Theme** · plantilla. Dawn (gratis) es buena base. Premium themes ($150-300) si quieres look específico.
3. **Producto** · sube 1 producto con foto + título + descripción + precio.
4. **Pago** · activa Shopify Payments + PayPal. Stripe Connect para UE.
5. **Envío** · define zonas, precios, plazos.

60 minutos bien usados. Después el tiempo va a contenido + ads + atención cliente.

## El ejemplo real

**Workflow Shopify setup 60 min · primera tienda**

### Fase 1 · Cuenta + dominio (10 min)

1. Ir a [shopify.com](https://shopify.com) → 14 días trial gratis.
2. Email + password + nombre tienda.
3. Settings → Domains → "Buy new domain" o "Connect existing".
4. Si compras nuevo en Shopify: $14/año.
5. Si conectas existente (Cloudflare/Namecheap): copias DNS records.

### Fase 2 · Theme (10 min)

1. Online Store → Themes → Dawn (gratis) ya viene.
2. Customize · cambia:
   - Logo (sube tu PNG transparente · M2.L6 si no lo tienes).
   - Color scheme · usa tu paleta marca.
   - Tipografía · 2 fuentes coherentes con brand kit.
3. Save.

### Fase 3 · Producto (15 min)

1. Products → Add product.
2. Title: ej. "Mate Térmico Runner 500ml".
3. Description: descripción que vende (3 párrafos · problema + solución + diferencial).
4. Media: 5-7 fotos producto (foto real + mockups IA si necesitas).
5. Price: precio final. Compare-at-price si quieres mostrar "antes/después".
6. Inventory: tracking opcional (si dropshipping con supplier integrado, off).
7. Shipping: weight + dimensions.
8. SEO snippet: title + meta description optimizados.
9. Save + Make Available "Online Store".

### Fase 4 · Pago (10 min)

1. Settings → Payments.
2. Shopify Payments → activa (KYC: DNI, cuenta bancaria, número fiscal).
3. PayPal → conecta cuenta PayPal Business.
4. Activos: tarjeta + PayPal + Bizum (España) si plan permite.

### Fase 5 · Envío (10 min)

1. Settings → Shipping and delivery.
2. Define zonas:
   - Spain → 4,90€ envío 24-48h · gratis a partir 40€.
   - Resto UE → 9,90€ · 5-7 días.
   - LATAM → 19,90€ · 10-15 días (si vendes internacional).
3. Save.

### Fase 6 · Test (5 min)

1. Order test propio · usa tarjeta personal · cobra y refund inmediato (Shopify lo permite).
2. Verifica que email confirmación llega.
3. Verifica que cliente puede completar checkout sin errores.

Total: 60 min. Tienda lista para vender al primer cliente.

## El prompt copiable

Checklist Shopify primera tienda (imprime):

```
☐ Trial Shopify activo (14 días gratis)
☐ Dominio propio conectado
☐ Theme personalizado (logo + colores + fuentes)
☐ 1 producto subido (título + descripción + 5 fotos + precio)
☐ Shopify Payments activo (DNI + banco verificados)
☐ PayPal conectado
☐ Zonas envío definidas + precios
☐ Email confirmación testado
☐ Checkout testado con tarjeta real (refund inmediato)
☐ Política de devolución redactada (14 días UE obligatorio)
☐ Política privacidad + cookies (legal LSSI España)
☐ Página About (quién eres · 100 palabras · cara humana)
```

12 items. Si no completas los 12 antes de abrir ads, los ads pierden eficacia (la tienda parece "amateur" al cliente).
