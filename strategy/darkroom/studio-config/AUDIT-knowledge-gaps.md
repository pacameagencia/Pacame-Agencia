Components of Good Video Scripts:

### D2 · Script Tips (líneas 1702-1712)

**FALTA**: 5 tips para scripts naturales.

```
1. KEEP IT NATURAL
   People talk conversationally, not formally

2. USE CONTRACTIONS
   "I've" not "I have"
   "can't" not "cannot"

3. SHORT SENTENCES
   Easier to understand and deliver

4. SPECIFIC DETAILS
   "Perfect temperature in 30 seconds" not "makes good coffee"

5. EMOTION
   Show enthusiasm, passion, authenticity
```

## §G · COMMON ISSUES + SOLUTIONS TROUBLESHOOTING (líneas 882-929)

**FALTA**: troubleshooting matrix completo.

```
ISSUE 1: FACE DOESN'T MATCH CONSISTENCY SHEET
Problem: Generated image shows different face/features
Solution:
  - Use more detailed consistency sheet reference
  - Add explicit facial description in prompt:
    "Asian woman, red hair, dramatic makeup, confident expression, exact face from consistency sheet"
  - Use Nano Banana Pro instead of other models
  - If still failing: Use prompt enhancer to strengthen prompt

ISSUE 2: BODY POSITION LOOKS UNNATURAL
Problem: Character looks awkward in pose
Solution:
  - Specify pose clearly: "Standing naturally, hands at sides, looking at camera"
  - Add context: "Professional pose, comfortable stance, natural positioning"
  - Generate 3 variations · choose best
  - Or regenerate with different pose instruction

ISSUE 3: CLOTHING DOESN'T MATCH SCENE
Problem: What character is wearing doesn't fit scenario
Solution:
  - Specify outfit in scenario prompt: "Girl wearing casual gym clothes, holding water bottle"
  - Or specify clothing change in integration prompt: "Girl in athletic wear for gym scene"
  - Add clothing detail to consistency sheet updates as needed

ISSUE 4: LIGHTING LOOKS WRONG
Problem: Lighting doesn't match scenario
Solution:
  - Specify lighting in base scene prompt: "Natural daylight from window, warm afternoon light"
  - Ensure consistency between base scene and integrated character
  - Integration model usually matches lighting automatically
```

### T2 · Cinematic vs Natural rendering shifts (líneas 4833-4866)

**FALTA**: 6 dimensiones cambio.

```
LIGHTING:
  Cinematic: carefully positioned, dramatic, intentional shadows
  Natural: flat, even, or directional (window/daylight)

COLOR GRADING:
  Cinematic: rich, saturated, color-corrected
  Natural: muted, realistic, whatever natural light produces

COMPOSITION:
  Cinematic: rule of thirds, dynamic angles
  Natural: simple framing, centered subject, casual angles

DETAIL LEVEL:
  Cinematic: every texture visible, hyper-detailed
  Natural: normal phone photo level

PERFECTION:
  Cinematic: everything perfect, no imperfections
  Natural: slight imperfections, natural variations

EQUIPMENT ASSUMPTIONS:
  Cinematic: AI assumes professional camera, lenses, rig
  Natural: AI assumes basic smartphone or consumer camera

The irony: BOTH approaches use the EXACT SAME AI MODEL.
The difference is entirely in the prompt language.
```
