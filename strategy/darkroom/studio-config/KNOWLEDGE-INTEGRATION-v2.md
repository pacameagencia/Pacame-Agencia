✓ Real highlights (oily zones forehead/nose, NOT plastic shine)
✓ Baby hairs + messy strands (perfect hair = fake)

### 2.8 NUEVO · Object & Material Texture Checklist 6 items (línea 3596-3611)

```
OBJECT TEXTURE CHECKLIST (pre-publish all product/vehicle shots):
✓ Micro-scratches and scuffs on plastics/metal
✓ Fingerprints/smudges on glossy surfaces
✓ Real fabric behavior (fold tension, stretch, stitching, pilling)
✓ Edge wear (corners worn, paint/chrome imperfections)
✓ Correct lighting response (matte vs glossy vs satin)
✓ Consistent contact shadows (object sits in scene, not floating)
```

**Similarity** (mismo system):
1. Create reference images from multiple angles
2. Generate consistency sheet

**Difference**:
- **Character Consistency**: focus on facial features, expression, presence — UNIQUE IDENTITY

**Why Object Consistency matters Dark Room**:
- Complex details (stitching, patterns, logos, text)
- Specific colors and finishes (metallic, matte, glossy)
- Brand authenticity (exact shoe design, handbag shape, packaging)
- Texture requirements (leather grain, fabric weave, metal finish)
