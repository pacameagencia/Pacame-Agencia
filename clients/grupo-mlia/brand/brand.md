- **Nombre:** Grupo M-LÍA (marca paraguas). Sub-marcas: **M-LÍA** (restaurante,
  eventos, catering, terraza, piscina, El Cielo), **MANÍA** (discoteca/pub),
  **Chocolate y Menta** (crepería).
- **Esencia:** "Creadores de experiencias. El punto de encuentro de referencia
  en Hellín." Grupo hostelero serio, multi-espacio, local.
- **Valores (manual):** Liderazgo · Proximidad · Compromiso · Modernidad.
- **Iconografía:** la **jirafa** (animal fetiche, preside la entrada) y el
  **sol** (energía, calidez). Usar como guiño sutil, nunca infantil.

operativa (derivada, solo para estados UI): `#0A0A0A` (negro suave fondo),
`#1A1A1A` (superficie), `#E8C766` (dorado claro hover), `#8C7020` (dorado
oscuro), `#9A9A9A` (texto secundario sobre negro).

## 3. Tipografía (manual sección 4)

- **Corporativa / titulares:** **Blantick Script** (`…\M-LIA\TIPOGRAFIA\
  Blantick Script.otf|ttf`). Script elegante — solo títulos de marca, hero,
  nombres de espacio. Nunca para párrafos.
- **Complementaria / texto:** **Montserrat** (Google Fonts, pesos 300/400/600/
  700). Todo el cuerpo, navegación, botones, formularios. Moderna y legible.
- Jerarquía web: H1/H2 marca → Blantick Script (o Montserrat 700 si legibilidad
  lo exige a tamaño pequeño); H3+ y texto → Montserrat.
- **Prohibido:** system-ui, Arial, fuentes random. Sin distorsionar Blantick.

## 4. Voz y tono (copy)

- Cercano y profesional. Hostelería seria que cuida cada detalle, sin pomposidad.
- Frases cortas, verbos activos, concreción local ("en Hellín", "tu día").
- Foco comercial: **eventos, bodas, comuniones, celebraciones de empresa**
  (prioridad del cliente) → beneficio + prueba + CTA claro (WhatsApp/reserva).
- **Prohibido (palabras IA):** desbloquea, embárcate, viaje, transformador, en
  última instancia, "no es solo X, es Y", adjetivos vacíos. Ver
  `docs/protocols/quality-gate.md`. Revisor: `quality-reviewer`.
