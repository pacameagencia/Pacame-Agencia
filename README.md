# PACAME Agencia

Agencia digital potenciada por IA. 7 agentes especializados + 1 humano.

## Estructura

- `web/` — Next.js 15 website (App Router, Tailwind, shadcn/ui)
- `agents/` — Definiciones de agentes IA (Nova, Pixel, Atlas, Nexus, Core, Pulse, Sage + DIOS)
- `brand/` — Identidad visual y guías de marca
- `strategy/` — Estrategias de SEO, growth, social media y negocio

## Stack

- **Frontend**: Next.js 15, React 19, Tailwind CSS, Framer Motion
- **Backend** (planned): Supabase, Stripe, Resend
- **Deploy**: Vercel
- **IA**: Claude API (Opus + Sonnet)

## Desarrollo

```bash
cd web
npm install
npm run dev
```
